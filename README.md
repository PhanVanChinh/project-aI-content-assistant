## 📖 Giới thiệu
**AI Content Assistant Pro** là một công cụ hỗ trợ sáng tạo nội dung mạnh mẽ dành cho các nhà sáng tạo nội dung, marketer và chủ doanh nghiệp. Ứng dụng sử dụng mô hình ngôn ngữ lớn **Gemini AI** của Google để phân tích các bài viết dài và tự động chuyển đổi chúng thành các định dạng nội dung tối ưu cho nhiều nền tảng mạng xã hội khác nhau.

## 🚀 Các tính năng chính
- 📝 **Tóm tắt thông minh:** Rút gọn các bài viết dài thành các ý chính súc tích chỉ trong vài giây.
- 📱 **Đa nền tảng (Multi-platform):**
  - **Facebook:** Tạo bài đăng thu hút với emoji và phong cách gần gũi.
  - **LinkedIn:** Tạo nội dung chuyên nghiệp, tập trung vào giá trị và kết nối.
  - **Twitter (X):** Tối ưu hóa thông điệp trong giới hạn ký tự ngắn gọn.
- 🔍 **Gợi ý từ khóa SEO:** Tự động đề xuất các hashtag và từ khóa liên quan để tăng khả năng hiển thị.
- ⚡ **Giao diện hiện đại:** Trải nghiệm người dùng mượt mà với hiệu ứng chuyển động và tính năng copy nhanh 1-click.

## 🛠️ Công nghệ sử dụng
- **Frontend:** React 19, TypeScript, Vite.
- **Styling:** Tailwind CSS 4, Lucide Icons.
- **Animation:** Framer Motion (Motion).
- **AI Engine:** Google Gemini AI SDK (`@google/genai`).

## 💻 Hướng dẫn cài đặt (Run Locally)

### Điều kiện tiên quyết
- Đã cài đặt **Node.js** (Phiên bản 18 trở lên).

### Các bước thực hiện
1. **Tải mã nguồn và cài đặt thư viện:**
   ```bash
   npm install
   ```

2. **Cấu hình API Key:**
   - Tạo file `.env` (hoặc chỉnh sửa `.env.local`) tại thư mục gốc.
   - Thêm dòng sau với API Key của bạn:
     ```env
     GEMINI_API_KEY=your_api_key_here
     ```


3. **Chạy ứng dụng:**
   ```bash
   npm run dev
   ```
